# Assets del Proyecto ATS Mobile

Esta carpeta contiene los recursos gráficos de la app.

## Imágenes necesarias (agregar manualmente)

- `icon.png` — Ícono de la app (1024x1024px)
- `splash.png` — Imagen del splash screen (1242x2688px)
- `adaptive-icon.png` — Ícono adaptativo para Android (1024x1024px)
- `favicon.png` — Favicon para web (48x48px)

> Los archivos de imagen no se incluyen en este repositorio.
> Agregarlos antes de compilar la app.
